Work

Projects & work

Shufflecast — High-Performance Optical Multicast Architecture

Rice University · 2018–2020

Designed an edge-restricted optical architecture that physically segregates multicast and unicast traffic, connecting ToR switches via passive optical splitters to deliver physical-layer multicast with no core resource consumption, no extra rack space, and sublinear latency growth. Built and evaluated a complete prototype across LDA, TPC-H, and Paxos workloads — achieving 6.04×–52.1× speedup in multicast throughput over MPI, up to 6.24× over state-of-the-art, and a 43.1% application-level runtime improvement.

C / C++ Optical Networking MPI HPC NS3

Compiler-Driven Network Optimization for HPC Applications

Rice University · 2020–2022

Designed an end-to-end automation pipeline that eliminates the expert burden of manually tuning reconfigurable datacenter networks. Extended the ROSE Compiler Framework (C++) to detect oscillating MPI communication patterns via CFG and dataflow analysis, mapped each pattern to optimal server placement, and injected reconfiguration code via source-to-source transformations — reducing application runtime by 40% with zero user expertise or manual code modification required.

C++ ROSE Compiler MPI Static Analysis Datacenter Networks

Automated Proof Mining & Synthesis for Program Verification

Rice University · 2015–2018

Implemented a proof mining system in Dafny/Z3 to extract verification patterns from existing verified programs, then extended Frama-C to synthesize correctness proofs via automated theorem proving and invariant generation. Significantly reduced manual verification effort by learning reusable proof strategies from corpora — applicable to validating correctness of latency-critical software components.

Dafny Z3 Frama-C Coq Formal Verification

Integrated OAuth 2.0 flows with encrypted token storage, architected a unified data model that translates a single input into multiple proprietary third-party formats, and added background scheduling with end-to-end integration tests.
